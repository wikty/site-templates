Inline code

Use <code> to wrap inline snippets of code

For example, <section> should be wrapped as inline.

Block code

Use <pre> for multiple lines of code.

You may optionally add the .pre-scrollable class, which will set a max-height of 340px and provide a y-axis scrollbar.

Use <kbd> to indicate input that is typically entered via keyboard.
