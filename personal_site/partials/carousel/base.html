Carousel

  • Carousels don’t automatically normalize slide dimensions. As such, you may need to use additional utilities or custom styles to appropriately size content.

  • The .active class needs to be added to one of the slides otherwise the carousel will not be visible.

  • Set a unique id on the .carousel for optional controls indicators, especially if you’re using multiple carousels on a single page.

  • Control and indicator elements must have a data-target attribute (or href for links) that matches the id of the .carousel element.

  • The data-ride="carousel" attribute is used to mark a carousel as animating starting at page load.
